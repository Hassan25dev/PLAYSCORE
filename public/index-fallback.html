PlayScore - Troubleshooting

Application Loading Issue

You're seeing this page because the PlayScore application is experiencing loading issues. This could be due to several reasons:

  • The Vite development server is not running or not accessible
  • There's a JavaScript error preventing the application from initializing
  • Browser cache or cookie issues
  • Network connectivity problems

Common Solutions

1. Restart the development servers

Make sure both the Laravel and Vite servers are running:

# Terminal 1
php artisan serve

# Terminal 2
npm run dev

2. Clear browser cache and cookies

Use our cache clearing tool or clear them manually in your browser settings.

3. Try a different browser

Sometimes browser-specific issues can cause loading problems.

4. Check for JavaScript errors

Press F12 to open developer tools, go to the Console tab, and look for error messages.
